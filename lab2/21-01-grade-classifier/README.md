# Lab02 练习 1 — 成绩等级判断器

## 考察内容

- `if / else if / else` 条件分支
- 比较运算符（`>=`、`<`、`>`）
- 逻辑运算符（`&&`、`||`）
- 分支条件的**顺序**对结果的影响
- 布尔返回值的表达方式

## 题目说明

实现 `GradeClassifier` 类中的三个方法，完成成绩分级、通过判断和 GPA 换算功能。

### 方法签名

```java
public static String classify(int score)
public static boolean isPassed(int score)
public static double toGPA(int score)
```

### classify — 评级规则

| 分数范围    | 返回值       |
|-----------|------------|
| 90 ~ 100  | `"优秀"`    |
| 75 ~ 89   | `"良好"`    |
| 60 ~ 74   | `"及格"`    |
| 0  ~ 59   | `"不及格"`   |
| 其他（< 0 或 > 100） | `"无效分数"` |

### isPassed — 通过判断

- 分数在 **0 ~ 100** 之间（含两端），且 **>= 60** 时返回 `true`
- 分数无效（< 0 或 > 100）时返回 `false`

### toGPA — GPA 换算

| 分数范围    | GPA  |
|-----------|------|
| 90 ~ 100  | 4.0  |
| 80 ~ 89   | 3.0  |
| 70 ~ 79   | 2.0  |
| 60 ~ 69   | 1.0  |
| 0  ~ 59   | 0.0  |
| 无效分数   | -1.0 |

## 完型填空说明

源文件 `GradeClassifier.java` 已提供方法骨架，需要填写以下空白处：

| 编号 | 位置 | 要填写的内容 |
|------|------|------------|
| 填空1 | `classify` 无效判断 | 将 `false` 替换为正确的无效分数判断条件 |
| 填空2 | `classify` 优秀判断 | 将 `false` 替换为 `score >= 90` |
| 填空3 | `classify` 良好判断 | 将 `false` 替换为 `score >= 75` |
| 填空4 | `classify` 及格判断 | 将 `false` 替换为 `score >= 60` |
| 填空5 | `isPassed` 返回值 | 将 `false` 替换为完整的布尔表达式 |
| 填空6a~6d | `toGPA` 各档返回值 | 将 `0.0` 替换为正确的 GPA 值 |

> **注意**：填空1 中的条件顺序很重要。先做无效判断，再做等级判断。
> 等级判断的顺序也很关键——如果把 `>= 60` 放在最前面会发生什么？
